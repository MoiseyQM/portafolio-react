# Portafolio Personal

Portafolio web de Juan Moisés Quispe Marcas desarrollado con React y TypeScript.

## Tecnologías

- **React 18**
- **TypeScript**
- **Vite**
- **CSS3**

## Instalación

```bash
git clone https://github.com/MoiseyQM/portafolio-react.git
cd portafolio-react
npm install
npm run dev
```
